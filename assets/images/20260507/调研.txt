市场继续上涨，算力相关方向依然是绝对主线，现在全球算力资本开支投入太大，很多领域都出现严重的缺货，中国供应链已经深入嵌入全球算力产业，AI发展对国内很多行业都是大利好。
海外算力有两个大新闻。
第一个是Antropic跟马斯克旗下的XAI达成一项合作，Anthropic租下XAI在美国田纳西孟菲斯的Colossus 1 数据中心全部算力，获得22万块英伟达GPU，用来支撑旗下Pro和MAX会员的算力供应，双方未来未来还准备开发GW级别的太空数据中心项目。
Colossus 1 数据中心24年3月开始建设，24年9月正式上线，属于马斯克的超级工程，只有了122天就顺利上线，包括14万块H100芯片，5万块H200芯片和3万块GB200芯片。XAI今年年初被并入SpaceX，一直发展不太顺利，在北美大模型中用户排名垫底，Colossus 1 数据中心的产能利用率估计比较低，这次正好租给Antropic提升产能利用率。
Antropic从25年下半年开始飞速发展，26年一季度增长进一步提速，目前年化收入达到440亿美金，大幅超越OpenAI的300多亿美金。但是在算力规模上，OpenAI布局比较早，通过跟微软和甲骨文的合作，已经获得了10GW算力，算力资源非常充足。
Antropic起步较晚，主要跟微软和谷歌合作，目前仅有1-2GW的算力，算力不足导致Antropic必须放弃很多低利润业务，包括不能把自己的套餐用于龙虾，高峰时段必须限速。
为了缓解算力不足，Anthropic今年4月紧急向亚马逊AWS和谷歌的采购了4-5GW的算力，同时本次又长期租赁了XAI的22万块GPU。本次租赁完成以后，Anthropic算力紧张局面得到很大缓解，直接宣布解除 Claude Code 和 Claude API 的速率限制，并且取消 Pro / Max 用户在高峰时段的暗中降速。
现在全球AI算力领域依然严重供不应求，GPU，存储，光互联，燃气轮机全面紧缺，每一个环节的需求都难得到满足。
第二个是北美光芯片龙头Lumentum召开了最新业绩交流会，受到激光器和光芯片短缺的影响，Lumentum的业绩表现小幅低于预期。目前光通信领域供需失衡非常严重，整体需求比产能多30%，目前长期订单已经签到2028年，核心客户包括英伟达、谷歌、微软、Meta，所欲欧订单都是长期不可撤销的订单，而不是短期订单。
光芯片的主要玩家都是美国和日本企业，效率低下，扩产速度慢，导致出现非常严重短缺局面，国内厂商大大受益，核心关注国内光芯片龙头东山精密。

黄金板块，最近央行购金力度开始增加。26年1-4月，央行购金分别是4万盎司，3万盎司，16万盎司和26万盎司，大约是1.2吨，0.9吨，5吨和8吨，随着3月以后黄金价格持续走弱，央行购金力度开始大幅提升。
当前中国央行的黄金持有量是7464万盎司，大约相当于2300吨，一吨黄金大约10.5亿，央行持有的黄金大约相当于2.4万亿人民币或者3500亿美金，大约是我们外汇储备的十分之一，这个体量不算太大。
在全球局势多变的大背景下，央行购金是一个长期趋势，也是支撑金价上涨的主要动力，黄金板块，黄金价格进一步下行空间有限。
铜板块，最近铜价也开始触底回升。铜这一轮本质上跟AI关联度非常高，铜未来最大需求增量是北美数据中心建设带来的电力用铜增长，包括电力传输的高压电缆，数据中心的铜缆，海上风电项目的海缆，都会带动铜需求的增长。
铜的供应增长非常受限，海外铜矿建设需要8-10年，海外大的铜矿都处于减产周期，只有国内紫金和洛钼提供一些产能增量。

铜和金板块未来都比较看好，可以布局紫金矿业，洛阳钼业和赤峰黄金。
