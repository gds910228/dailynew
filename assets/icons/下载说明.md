## 🎨 推荐下载网站

### 方案1：阿里巴巴 iconfont（最推荐）

**网址：** https://www.iconfont.cn/

**优点：**
- 国内网站，访问速度快
- 图标数量多，风格统一
- 可以下载PNG格式
- 完全免费

**下载步骤：**

1. **访问网站** https://www.iconfont.cn/

2. **搜索图标**（搜索以下关键词）：
   - 首页/我的：搜索 "home", "user", "profile"
   - 功能：搜索 "search", "close", "download", "heart", "share", "play"
   - 状态：搜索 "empty", "delete"

3. **选择图标**：
   - 点击喜欢的图标
   - 选择"下载"按钮
   - 格式选择：PNG
   - 尺寸选择：64x64 或 81x81（TabBar图标）
   - 颜色设置：
     - 未选中图标：`#999999` 或 `#666666`（灰色）
     - 选中图标：`#667eea`（紫色）
   - 点击下载

4. **重命名文件**：
   下载后按照上述清单重命名文件

---

### 方案2：Remix Icon（国外，但更现代）

**网址：** https://remixicon.com/

**优点：**
- 图标风格现代、统一
- 线性图标，符合设计规范
- 直接下载，无需注册

**下载步骤：**

1. 访问 https://remixicon.com/

2. 搜索需要的图标：
   - Home line → home.png
   - User line → profile.png
   - Search line → search.png
   - Close line → close.png
   - Download line → download.png
   - Heart line → heart.png
   - Share line → share.png
   - Play line → video.png
   - Delete bin line → delete.png

3. 点击图标 → 点击下载按钮
4. 下载SVG或PNG格式
5. 使用在线工具调整颜色和尺寸
