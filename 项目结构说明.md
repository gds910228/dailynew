# 项目文件结构说明

## 📁 项目结构

```

│   ├── app.js            # 小程序入口
│   ├── app.json          # 小程序配置
│   └── app.wxss          # 全局样式
│
├── 🛠️ 发布工具

│   └── generate_icons.py       # 图标生成脚本
│
├── 📊 数据文件
│   └── data/
│       ├── articles.json      # 文章数据（提交到GitHub）
│       └── README.md         # 数据结构说明
│
├── 🎨 Web管理后台
│   └── admin/
│       ├── index.html         # 后台界面（支持多图URL）
│       ├── app.js             # 后台逻辑

├── 📚 文档

│   ├── 快速入门.md              # ⭐ 5分钟上手
│   ├── 每日维护指南.md          # ⭐ 日常维护流程

│   ├── DEPLOYMENT.md            # 部署说明
│   ├── CLAUDE.md                # Claude使用说明
│   └── archive/                 # 归档文档

    └── .claude/                # Claude配置
```

## 🚀 快速开始指南

### 第一次使用

1. **配置Web后台**
   ```
   编辑 admin/config.js
   填写 GitHub Token 和用户名
   ```

2. **发布第一篇文章**
   ```bash
   # 1. 准备图片，放到 assets/images/
   # 2. 运行发布工具
   双击：publish-simple.bat

   # 3. 填写文章
   Web后台会自动打开
   复制URL → 填写信息 → 提交
   ```

3. **查看效果**
   ```
   微信开发者工具 → 编译
   下拉刷新查看文章
   ```

## 📝 文件命名规范

### 工具文件
- 使用英文文件名
- 小写+连字符
- 示例：`publish-simple.bat`, `generate-urls.bat`

### 图片文件
- **推荐**：英文文件名
  - 示例：`20250119001.jpg`, `gl-17-openai.jpg`
- **支持**：中文文件名
  - 工具会自动URL编码
  - 示例：`GL-17号-OpenAI营销.jpg`
